# Paulkirchhoff Notw

## How do I install these formulae?

`brew install paulkirchhoff/notw/<formula>`

Or `brew tap paulkirchhoff/notw` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "paulkirchhoff/notw"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
